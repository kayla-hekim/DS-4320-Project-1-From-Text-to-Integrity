# DS 4320 Project 1: From Text to Integrity

### Executive Summary

This repository contains a Data by Design project 1, focused on early detection of whether textual news-style articles were  written by humans or artificial intelligence (AI) generated. The project utilizes a dataset from a Kaggle competition called "LLM - Detect AI Generated Text". This repository also includes necessary images, supporting documentation, metadata tables, a press release, MIT License, and pipeline jupyter notebooks that handle ppreprocessing and model training on the data. The project's repository demonstrates the acquisition of the data, preprocessing the data, exploratory data analysis (EDA) on the data, and binary classification methods to predict if a text is AI-generated or completely human-authored.
